# docs: 프로젝트 문서 허브

> [!NOTE]
> All AI agent interactions with this documentation suite must comply with the [Agent Governance Hub](./00.agent-governance/README.md).

`docs/`는 요구사항, 설계, 실행, 운영, 사고/회고를 숫자형 taxonomy로 관리하는 저장소 문서 허브입니다.

## 문서 사용 원칙

- 사람 중심 개요 문서와 루트 가이드는 한국어로 작성합니다.
- AI Agent가 직접 실행에 사용하는 거버넌스/운영 규칙 문서는 영어로 작성합니다.
- 특히 `docs/00.agent-governance/`는 영어 전용입니다.

## 문서 생명주기

`00.agent-governance -> 01.prd -> 02.ard/03.adr -> 04.specs -> 05.plans -> 06.tasks -> 07.guides -> 08.operations -> 09.runbooks -> 10.incidents -> 11.postmortems`
