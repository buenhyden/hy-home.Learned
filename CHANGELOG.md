# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-01-27

### Added

- **Documentation**: Comprehensive `docs/` folder reorganization.
  - Added `docs/index.md` as the central knowledge hub.
  - Added `docs/development-setup-guide.md` with strict `uv` workflows.
  - Added `docs/knowledge-management.md` defining the TIL pipeline.
- **Infrastructure**: Added `.github/workflows/labeler.yml` for auto-triaging PRs.
- **Governance**: Added `CODEOWNERS` and updated `LICENSE` (MIT).

### Changed

- **Config**: Optimized `.github/dependabot.yml` to remove invalid pre-commit hooks.
- **Mission**: Clarified project vision in `docs/project-mission.md` for "Vision 2026".
