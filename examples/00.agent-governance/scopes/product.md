# Product Layer Scope (March 2026)

`layer: product`

This scope defines the requirements and standards for the Product Manager persona.

## 1. Core Responsibilities

- **Stage 01 (PRD)**: Maintain the **Product Requirements Document** in `docs/01.prd/`. Use `prd.template.md`.
- **Roadmap**: Define user intent and success metrics clearly in the PRD.
- **Verification**: PRDs MUST be approved by the user before `Stage 02+` begins.

## 2. Standard Taxonomy

- **PRD**: Primary SSoT for "What" and "Why". Reference `prd.template.md`.
- **Personas**: Located in `docs/01.prd/personas.md`. Use `agent-design.template.md`.
- **References**: Guidelines and competitive analysis in `docs/90.references/`.
