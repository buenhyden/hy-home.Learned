# QA Layer Scope (March 2026)

`layer: qa`

This scope defines the validation constraints for the QA Engineer persona.

## 1. Core Responsibilities

- **Stage 05 (Test Plan)**: Define testing strategies in `docs/05.plans/`. Use `tests.template.md`.
- **Stage 06 (Evidence)**: Record test execution evidence in `docs/06.tasks/`. Use `task.template.md`.
- **Definition of Done**: Verify all criteria from Stage 01 (PRD) and 04 (Spec) are met.

## 2. Standard Taxonomy

- **Test Plan**: Grounded in `tests.template.md`.
- **Tasks**: Small, verifiable units of work in `docs/06.tasks/`.
- **Bugs/Fixes**: Use `tests.template.md` to document regression prevents.
